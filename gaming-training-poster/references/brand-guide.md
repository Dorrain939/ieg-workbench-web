# Brand Guide · 游戏大厂培训品牌指南

> 这份文件是 Skill 的"灵魂"。每次生成海报前必读。

## 1. 品牌定位

我们是**游戏大厂的人才发展中心**，海报既要承袭"游戏感"（关卡、等级、像素、燃），又要保留"专业培训"的克制（信息清晰、便于查阅、不喧宾夺主）。

一句话：**"游戏皮肤 + 培训内核"**——视觉燃，但信息要正经。

## 2. 核心调性关键词（强制注入 prompt）

### 必含（任一场景）
- `game-inspired`, `pixel art accents`, `level-up motif`
- `clean composition`, `professional training context`
- `sense of progression`, `community vibe`

### 禁用（必须放进 Negative Prompt）
- `cartoon childish`, `anime fanart`, `gore`, `dark horror`
- `cluttered`, `messy typography`, `lens flare overload`
- `kawaii overload`（避免低龄化）
- `corporate stiff`, `stock photo style`（避免无聊化）

## 3. 调性平衡尺（每个场景在尺上找位置）

```
克制 ←─────────────●─────────────→ 燃
         S2 S5    │    S1 S3 S4 S6
                  └ 默认中点
```

| 场景 | 燃度 | 主导情绪 |
|---|---|---|
| S1 新人训练营 | 7/10 | 朝气、期待、启航 |
| S2 领导力 | 4/10 | 沉稳、远见、力量 |
| S3 技术分享 | 7/10 | 极客、专注、未来 |
| S4 文化活动 | 8/10 | 温暖、欢聚、节日 |
| S5 晋升表彰 | 5/10 | 荣耀、仪式感、克制 |
| S6 内部赛事 | 9/10 | 对抗、高燃、电竞 |

## 4. 视觉元素优先级（从高到低）

1. **主标题**（必须最显眼）
2. **Hero 元素**（场景标志物：飞船 / 勋章 / 代码雨 / 烟花 / 奖杯 / 计分板）
3. **Logo**（右上或左上，永远在安全区）
4. **副标题 / 关键信息**
5. **日期 · 地点 · 主办方**
6. **二维码**（必须独立放置，不要叠加纹理）
7. **装饰元素**（像素飞溅、光效、几何线条）

## 5. 内容/价值观红线

- 不出现真实玩家头像、未授权 IP（包括公司自家产品 IP，需走授权流程）
- 不出现具体业绩数字（除非用户明确提供并核对）
- 不出现可被理解为"加班光荣"的语义
- 不使用比较级贬损其他职级/团队
- 培训≠考核：避免"末位淘汰""强制""必须达标"等措辞

## 6. 一致性检查清单

每张海报交付前，对照本清单：

- [ ] 主色占比 ≥ 30%（来自 design-system 的场景主色）
- [ ] 至少包含 1 个游戏化元素（关卡、等级、勋章、像素、计分板……）
- [ ] 文字层级 ≥ 3 层（标题 / 副标题 / 信息）
- [ ] Logo 清晰可辨，无变形
- [ ] 信息完整：标题、日期、地点、主办、报名通道（如适用）
- [ ] 调性与场景燃度尺一致
