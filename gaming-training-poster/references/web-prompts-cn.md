# 网页端中文 Prompt 卡片（零 API 路线）

> 适用于：豆包/即梦、文心一格、通义万相、Midjourney 网页版、Bing Image Creator 等。
> 用法：复制对应场景的 prompt → 粘贴到网页工具 → 出图 → 保存到 `assets/samples/<scene>_bg_v1.png` → 跑 `compose_poster.py` 合成。
> 关键：**所有 prompt 都明确要求"画面顶部 35% 留空、底部 25% 留空、不要任何文字"**——这是给后期合成留出标题区和信息区的关键。

---

## S1 · 新人训练营（Onboarding / Bootcamp）

**中文 Prompt：**
```
竖版游戏公司新人训练营海报背景，主体是像素风火箭造型的游戏手柄从像素地形向上发射，
周围漂浮着"Lv.1"等级标记和柔和发光的经验值点，等距视角，整体传达启程感与成长感，
配色：暖色黎明天空过渡到深邃蓝色，丰富的像素艺术细节，专业排版感。
重要：画面顶部 35% 区域保留为简洁天空、底部 25% 保留为柔和渐变地面（用作后期文字叠加），
画面中绝对不出现任何文字、字母、汉字、Logo、UI 按钮、人物面部，
高清 8K，电影级布光，3A 游戏工作室美术风格。
```
**尺寸建议：** 2480×3508（A3）或 1240×1748（A4） · **风格关键词：** 像素艺术 / 等距 / 启程感

## 通用质量提示词（追加到任何场景末尾）

如果出图质感不够，在 prompt 末尾追加：
```
，无文字水印，无品牌标识，构图均衡，留白充足，超高清，专业海报级品质
```

## 常见网页工具差异提示

| 工具 | 中文 prompt | 尺寸控制 | 备注 |
|---|---|---|---|
| **豆包/即梦** | ✅ 优秀 | 选 9:16 / 3:4 | 国内首选，对中文场景理解最好 |
| **通义万相** | ✅ 优秀 | 选 3:4 | 阿里出品，质感稳定 |
| **文心一格** | ✅ 良好 | 选 3:4 | 风格偏写实 |
| **Midjourney** | ⚠️ 建议翻成英文 | `--ar 3:4` | 质感最高，但中文 prompt 效果一般 |
| **Bing Image Creator** | ⚠️ 建议英文 | 默认 1:1 需后期裁切 | 免费但尺寸不灵活 |

> 翻成英文请用 `references/scene-prompts.md` 里对应的英文版本——那是经过构图术语调优的。
